# 2SGamer Tool v1.0.0.0

Tool2SGamer là công cụ chạy ở chế độ Console + GUI, hỗ trợ thao tác và quản lý game 2SGamer.

Author: Yisu  
Company: 2SGamerCMD  
Version: 1.0.0.0  

---

## Giới thiệu

2SGamer Tool cung cấp một hệ thống tiện ích gồm:

- Thay đổi tiêu đề cửa sổ game
- Bộ công cụ hỗ trợ gameplay (Speed, Nitro, Freeze...)
- Quản lý nhiều cửa sổ game cùng lúc

Tool hoạt động với quyền Administrator và tương tác trực tiếp với tiến trình game.

---

## Tính năng chính

### 1. Hệ thống License

- Lấy HWID máy
- Mã hóa và gửi về server xác thực
- Tự động kiểm tra định kỳ
- Hiển thị trạng thái:
  - OK
  - PENDING
  - INVALID
  - ERROR

Chỉ khi License hợp lệ mới vào được menu chính.

---

### 2. Đổi Title Game

- Quét các cửa sổ game đang mở
- Chuẩn hóa lại tên hiển thị
- Hỗ trợ nhiều cửa sổ cùng lúc

---

### 3. Hack GUI

Giao diện điều khiển trực quan gồm:

- Speed Hack
- Multi Speed
- Max Nitro
- Freeze Boss
- Chạy trước 3 giây
- Follow NPC
- Auto F5

Hỗ trợ:

- Load và chọn nhân vật
- Attach đúng tiến trình
- Điều khiển realtime
- Hotkey điều khiển nhanh

---

### 4. Redirect 2SGamer

Chức năng:

- Theo dõi tiến trình khởi động
- Tự động đóng tiến trình gốc
- Có thể dừng redirect bất cứ lúc nào

Hoạt động liên tục cho đến khi nhập lệnh `stop`.

|------|-----------|
